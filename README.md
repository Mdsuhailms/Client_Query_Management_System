# Client_Query_Management_System
-> DESCRIPTION:
          "Web Application for Submitting and Managing queries.Using python and Streamlit."

-> FEATURES:
          1. Query submiting platform for clients.
          2. Managing queries through Support Dashboard.
          3. Queries accessed through Database.

-> Tools Used:
          1. Python, Pandas
          2. PosgreSQL (for database)
          3. Streamlit (for Frontend UI)

-> Install tools mentioned in requirements.txt file..
          using command `pip install`

-> Database connection to local server using `psycopg2` to connect with POSTGRESQL database.

-> To Run the application 
          Use the command `streamlit run app.py`
